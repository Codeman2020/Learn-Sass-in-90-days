# 2015年Sass課程資訊
個人將在2015年五月於北中南開設Sass課程，
今年與去年變動差異較大的是第二天課程，
課程走向主要針對「網頁版型管理」進行較透徹的講解，
同時介紹Fire.app erb template版型系統，
透過它來了解後端版型邏輯，並清楚理解如何與後端工程師合作，
第二天最後一小時也會分享自己現階段的前端開發心法，希望能讓大家滿載而歸:D
